## 7주차(주특기 심화 - React)

### **repeat**
- 문자열을 반복한 값을 반환하는 메서드

### **문법**
```js
string.repeat( count )
```

### **예제**
```js
'abc'.repeat( 2 ) // abcabc
```

```jsx
Home = () => {
  const num1 = 5;

  return (
    <div>
    <span>{"⭐️".repeat(3)}</span>
    <br />
    <span>{"⭐️".repeat(num1)}</span>
    </div>
  )
}

export default Home; 

// ⭐️⭐️⭐️
// ⭐️⭐️⭐️⭐️⭐️
```
