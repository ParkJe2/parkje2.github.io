### 3주차(프로그래밍 기초)

개인 프로젝트 준비 과정 중 알게된 **"toUpperCase()", "toLowerCase()"**에 대한 내용에 대해 정리해보기!

## **toUpperCase()**

```js
String.toUpperCase()
- 문자열을 대문자로 변환 후 반환한다.
- 원본 문자열은 변경되지 않는다
```

### **예제)**

```js
let str = "Hello toUpperCase";
str.toUpperCase(); // HELLO TOUPPERCASE
```

## **toLowerCase()**

```js
String.toLowerCase()
- 문자열을 소문자로 변환 후 반환한다.
- 원본 문자열은 변경되지 않는다
```

### **예제)**

```js
let str = "Hello toUpperCase";
str.toUpperCase(); // hello touppercase
```
