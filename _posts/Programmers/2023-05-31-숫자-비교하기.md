## 문제 설명

> 정수 num1과 num2가 매개변수로 주어집니다.<br>
> 두 수가 같으면 1 다르면 -1을 return 하도록 solution 함수를 완성해주세요.
> {: .prompt-info }

```js
function solution(num1, num2) {
  let answer = 0;
  if (num1 === num2) {
    answer = 1;
  } else {
    answer = -1;
  }
  return answer;
}
```
