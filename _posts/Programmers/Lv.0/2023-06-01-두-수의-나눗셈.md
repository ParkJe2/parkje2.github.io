## 문제 설명

> 정수 num1과 num2가 매개변수로 주어집니다.<br>
> num1을 num2로 나눈 값에, 1,000을 곱한 후 정수 부분을 return 하도록 solution 함수를 완성해주세요.
> {: .prompt-info }

```js
function solution(num1, num2) {
  return Math.floor((num1 / num2) * 1000);
}
```

### **Math.floor()**

소수점 이하를 버림한다.

### **Math.cell()**

소수점 이하를 올림한다.

### **Math.round()**

소수점 이하를 반올림한다.
