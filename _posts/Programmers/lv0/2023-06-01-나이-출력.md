## 문제 설명

> 머쓱이는 40살인 선생님이 몇년도에 태어났는지 궁금해졌습니다.<br>
> 나이 age가 주어질 때, 2022년을 기준 출생 연도를 return 하는 solution 함수를 완성해주세요.
> {: .prompt-info }

```js
function solution(age) {
  let answer = 2022 - age + 1;
  return answer;
}
```
