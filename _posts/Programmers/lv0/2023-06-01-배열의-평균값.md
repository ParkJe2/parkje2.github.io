## 문제 설명

> 정수 배열 numbers가 매개변수로 주어집니다.<br>
> numbers의 원소의 평균값을 return 하도록 solution 함수를 완성해주세요.
> {: .prompt-info }

```js
function solution(arr) {
  let answer = 0;

  for (let i = 0; i < arr.length; i++) {
    //arr의 길이만큼 반복
    answer += arr[i]; //answer에 arr의 모든 합
  }

  answer = answer / arr.length; //평균 구하기

  return answer;
}
```
