## 문제 설명

> 정수 n이 주어질 때, n이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 완성해주세요.
> {: .prompt-info }

```js
function solution(n) {
  let answer = 0;
  for (i = 0; i <= n; i++) {
    if (i % 2 == 0) answer += i;
  }
  return answer;
}
```
