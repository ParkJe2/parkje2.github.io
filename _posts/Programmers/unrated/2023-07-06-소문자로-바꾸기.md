## 문제 설명

<p>알파벳으로 이루어진 문자열 <code>myString</code>이 주어집니다. 모든 알파벳을 소문자로 변환하여 return 하는 solution 함수를 완성해 주세요.</p>

<hr>

### 제한사항

<ul>
<li>1 ≤ <code>myString</code>의 길이 ≤ 100,000

<ul>
<li><code>myString</code>은 알파벳으로 이루어진 문자열입니다.</li>
</ul></li>
</ul>

## 문제 풀이

```js

}
```
